### Заголовок
Типы данных: объекты

### Описание задания
"Ваша задача - найти какое количество раз текст в PATTERN повторяется в TEXT.
Повторения могут пересекаться друг с другом.
Предлагаем циклом перебрать все возможные позиции в TEXT в которых может начинаться PATTERN.
При переборе каждую позицию проверять циклом, перебирая символы в PATTERN и символы, которые идут начиная с перебираемой позиции.
Если при проверке хотя бы один из символов не совпадёт - позиция не подходит и переходим к следующей. В противном случае нужно увеличить счётчик повторений count. "

### Ожидаемый ответ

### Подсказка
"Для того чтобы узнать какой символ находится на i-й позиции в строке, используйте метод charAt(i), который есть у каждого объекта типа String.

При вынимании символов из строки не забудьте обратить внимание на то, что вы не выходите за границы этой строки 

Для проверки позиции на то что с неё начинается совпадение с PATTERN, вы можете завести булевский флаг, который после цикла проверки совпадения будет либо true (есть совпадение), либо false если хотя бы один из символов не совпал"
